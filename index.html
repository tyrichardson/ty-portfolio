SVC Data Management

This is an updated data version of the reporting tool used by Sexual Violence Center to aggregate their data for Federal and County grant reporting. It was built as a group project by me and four other student developers using React/Redux/Material UI-core and Node/Express/PostgreSQL. I recently update/refactored Passport for this demo. It is hosted on Heroku. Login with ty/ty for admin access and fran/fran for data-entry only access. When running reports, select date ranges between 10-01-2016 and 12-31-2017. When using Search/Update, use a confirmation number between 102 and 6,513.
